\solution{1}  We solve the problem using Fibonacci numbers:

\begin{pseudocode}
    \Function{fib}{$n$}:
    \State \# input: int $n > 0$
    \State \# output: fibonacci($n$)

    \If{$n > 1$}
        \State \Return $1$
    \Else
        \State \Return \textsc{fib}($n-1$) + \textsc{fib}($n-2$)
    \EndIf
    \EndFunction
\end{pseudocode}

The correctness is obvious and the complexity analysis is elusive.
